Detailed Description

template<typename R, typename... As>
class fn2::Function< R(As...)>

Function is an invocable object wrapper.

Function can wrap any invocable object, including regular function pointers, function-like objects (functors), lambda expressions, member function pointers, and member data pointers.

Function can have no wrapped object, in which case it is undefined behavior to attempt to invoke that Function. Users can query whether a Function has a wrapped object by using the Function::operator bool() member function, which returns false if there is no wrapped object.

If the wrapped object is small enough and has a suitable alignment, it will be stored inside the Function object without dynamic allocation. Otherwise, the wrapped object will be stored on the free store and handled via operator new()/operator delete().
